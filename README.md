# Vacuum World

Honors AI Agent Project. ([`DEMO`](https://ian13456.github.io/vacuum-world/)))

## Installation

```bash
git clone https://github.com/ian13456/vacuum-world.git

cd vacuum-world

npm i serve -g

serve -p 3000
```

Visit `localhost:3000`.
